Copy the following csv files to your file system when using dumbo.
Police_officials_monthly.csv
Poverty_rate_Census.csv
Surveillance_Camera.csv
Unemployment_Rate_Monthly.csv

Run the Spark_SQL_Implementation.txt file in pyspark

Will get an output csv file from each implementation. Copy each file to the local system.

Using this output-file, run the R file on your pc using R-studio, the end result is a png-image file and the correlation-coefficient.
